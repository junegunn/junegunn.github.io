# [0.70.0][0.70.0]

(Mar 2026)

[0.70.0]: https://github.com/junegunn/fzf/releases/tag/v0.70.0

## Quick summary

This release focuses on performance improvements and adds a couple of new
actions for dynamically changing display options.

| Type        | Class   | Name                  | Description                                       |
| :--         | :--     | :--                   | :--                                               |
| Enhancement | Perf    |                       | 1.3x to 1.9x faster filtering                    |
| New         | Action  | `change-with-nth`     | Dynamically change `--with-nth` option             |
| New         | Action  | `change-header-lines` | Dynamically change `--header-lines` option         |

## 2. New actions

### `change-with-nth`

<video src="https://github.com/user-attachments/assets/40abe7e6-8c98-4a13-85b7-17e69372ef9e" controls="controls" style="max-width: 100%"></video>

You can now dynamically change the `--with-nth` option using the
`change-with-nth` action. Requires `--with-nth` to be set initially. Multiple
options separated by `|` can be given to cycle through.

```sh
echo -e "a b c\nd e f\ng h i" | fzf --with-nth .. \
  --bind 'space:change-with-nth(1|2|3|1,3|2,3|)'
```

### `change-header-lines`

Similarly, `change-header-lines` lets you dynamically change the
`--header-lines` option.
